# AI Lead Qualification Automation

AI-powered lead qualification workflow built with Make.com, Groq AI, Google Sheets, and Gmail.

## Features

* Automatic lead qualification
* Lead scoring (Hot, Warm, Cold)
* Google Sheets integration
* Owner email notification
* Customer auto-reply

## Workflow

Webhook → Groq AI → JSON Parser → Google Sheets → Gmail

## Tech Stack

* PHP Native
* Vanilla JavaScript
* CSS Native
* Make.com
* Groq AI
* Google Sheets
* Gmail
